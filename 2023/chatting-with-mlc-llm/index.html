Chatting with MLC LLM

September 2, 2023 - One minute read - 172 words

Machine Learning Compilation for LLM (MLC LLM) is a high-performance universal deployment for large-language models.

Setup

Install TVM locally through pip:

pip install apache-tvm

Check TVM build option:

python3 -c "import tvm; print('\n'.join(f'{k}: {v}' for k, v in tvm.support.libinfo().items()))"

Install conda onto wsl:

wget https://repo.anaconda.com/miniconda/Miniconda3-py39_4.12.0-Linux-x86_64.sh
sh ./Miniconda3-py39_4.12.0-Linux-x86_64.sh

Updates conda

# update conda
conda update --yes -n base -c defaults conda
# install `conda-libmamba-solver`
conda install --yes -n base conda-libmamba-solver
# set it as the default solver
conda config --set solver libmamba

Validates conda installation:

conda info | grep platform

Installs Vulkan SDK

VulkanSDK-1.3.261.1-Installer.exe

To validate, install:

sudo apt-get install vulkan-tools

# Runs below command to check GPU information
vulkaninfo

Try out MLC Chat

To run the models on WSL, we try out the CLI version of MLC LLM:

Create new conda environment:

conda create -n mlc-chat-venv -c mlc-ai -c conda-forge mlc-chat-nightly
conda activate mlc-chat-venv

conda install git git-lfs
git lfs install

mkdir -p dist/prebuilt
git clone https://github.com/mlc-ai/binary-mlc-llm-libs.git dist/prebuilt/lib

cd dist/prebuilt
git clone https://huggingface.co/mlc-ai/mlc-chat-Llama-2-7b-chat-hf-q4f16_1

Runs the MLC Chat:

cd ../..
mlc_chat_cli --model Llama-2-7b-chat-hf-q4f16_1
